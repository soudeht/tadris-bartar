# Design Style Guide for "گوسفند یا چوپان"

## Design Philosophy

### Visual Language
- **Gen Z Aesthetic**: Bold, expressive, and unapologetically vibrant
- **Persian Cultural Fusion**: Modern digital design honoring Persian typography traditions
- **Educational Impact**: Using visual excitement to engage users in critical thinking
- **Dark Mode First**: Neon colors pop against dark backgrounds, reducing eye strain

### Color Palette - "Dopamine Neon"
- **Electric Blue**: #43a9dd - Primary action color, trust and technology
- **Neon Green**: #00ff00 - Success, learning, positive actions
- **Bright Pink**: #ff007f - Energy, passion, attention-grabbing
- **Gen-Z Yellow**: #f7d000 - Optimism, highlights, call-to-actions
- **Dark Background**: #0a0a0a - Deep black for neon contrast
- **Text Light**: #ffffff - High contrast readable text
- **Text Dark**: #1a1a1a - For light mode alternatives

### Typography
- **Primary Font**: Vazir (Persian) - Clean, modern, highly readable
- **Secondary Font**: Shabnam (Persian) - Elegant for headings and emphasis
- **Fallback**: Tahoma, system fonts for compatibility
- **English Font**: Poppins for Latin text elements

### Visual Hierarchy
- **Headings**: Large, bold Persian text with neon color accents
- **Body Text**: Smaller Vazir font with excellent readability
- **Interactive Elements**: Glowing neon effects on hover
- **RTL Layout**: Full right-to-left reading experience

## Visual Effects & Animation

### Core Libraries Used
1. **Anime.js** - Smooth micro-interactions and button animations
2. **p5.js** - Interactive sheep herd simulation and particle effects
3. **ECharts.js** - Data visualization for quiz results and statistics
4. **Splide.js** - Story carousel and image galleries
5. **Matter.js** - Physics-based animations for interactive elements
6. **PIXI.js** - Advanced visual effects and shader backgrounds
7. **Shader-park** - Dynamic gradient backgrounds and neon glows
8. **glfx.js** - Image processing and filter effects

### Animation Strategy
- **Scroll Animations**: Reveal content with staggered fade-ins
- **Hover Effects**: Neon glow expansion and color morphing
- **Loading States**: Animated progress with Persian cultural patterns
- **Micro-interactions**: Button press feedback with sound-like visuals

### Background Effects
- **Aurora Gradient Flow**: Dynamic color transitions across sections
- **Neon Grid Overlay**: Subtle tech-inspired pattern elements
- **Particle Systems**: Floating geometric shapes responding to scroll
- **Shader Backgrounds**: Liquid-metal displacement effects

### Interactive Components Styling
- **Chat Interface**: Holographic bubble design with typing indicators
- **Quiz Elements**: Progress bars with sheep-to-wolf transformation
- **Story Cards**: Lift and glow effects on hover with Persian patterns
- **Navigation**: Sticky header with neon underline active states

### Responsive Design
- **Mobile First**: Touch-friendly interactions (44px minimum)
- **Tablet Optimization**: Balanced layout between mobile and desktop
- **Desktop Enhancement**: Full visual effects and animations
- **High DPI**: Crisp visuals on retina displays

### Accessibility Considerations
- **High Contrast Mode**: Alternative color scheme for visual impairments
- **Reduced Motion**: Respect user preferences for animation
- **Keyboard Navigation**: Full tab order and focus indicators
- **Screen Reader**: Proper Persian text markup and ARIA labels

### Cultural Design Elements
- **Persian Patterns**: Subtle geometric motifs in backgrounds
- **Calligraphic Accents**: Decorative elements inspired by Persian script
- **Color Symbolism**: Respectful use of colors in Persian culture
- **Typography Rhythm**: Natural Persian text flow and spacing
